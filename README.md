# AI News Aggregator - Public Website

Daily digest of AI developments powered by AATF and Claude Opus 4.5.

## Live Site

**URL:** https://sparktsao.github.io/ai-news-aggregator-pub/

## What's This?

A simple, single-page HTML application that displays AI news from multiple sources:
- **News:** Product launches, company announcements
- **Research:** arXiv papers, research blogs
- **Social:** Twitter/Bluesky/Mastodon discussions
- **Reddit:** Community discussions
- **Jobs:** AI/ML job postings

## Features

✅ **Simple & Fast** - One HTML file, no build process
✅ **Date Selector** - Browse 28+ days of news
✅ **Category Filters** - Filter by news type
✅ **Vote Buttons** - Like/dislike functionality (ready for backend)
✅ **Responsive** - Works on mobile & desktop

## Technology

- **Frontend:** Plain HTML + CSS + Vanilla JavaScript (494 lines)
- **Data:** Static JSON files
- **Hosting:** GitHub Pages (free)
- **Updates:** Automated daily via pipeline

## Adding Voting Backend

To enable the voting buttons:

1. Deploy Cloudflare Worker from parent repo: `UI/worker/`
2. Add voting service files to this repo
3. Connect vote buttons to API

See parent repo for details: `ai-news-aggregator` (private)

## Data Updates

This site is automatically updated daily with fresh AI news. Data is generated by an automated pipeline and published here.

## How It Works

1. User visits `index.html`
2. JavaScript fetches `data/index.json` to get available dates
3. User selects date and category
4. JavaScript fetches data from `data/YYYY-MM-DD/*.json`
5. Displays in table format
6. Vote buttons ready for backend integration
